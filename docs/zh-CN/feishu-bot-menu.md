# 飞书机器人常置命令（自定义菜单）

飞书支持在**单聊**输入框上方固定机器人菜单，相当于「常置命令栏」。

## 配置步骤

1. 打开 [飞书开放平台](https://open.feishu.cn/app) → 你的应用 → **机器人**
2. 开启 **机器人自定义菜单** → 编辑
3. 展示样式建议选 **悬浮菜单**（7.22+ 客户端）
4. 添加菜单项，推荐两种方式：

### 方式 A：发送文字（最简单）

| 菜单名称 | 动作 | 发送文字 |
|---------|------|----------|
| 状态 | 发送文字 | `/status` |
| 续聊 | 发送文字 | `/resume` |
| 新会话 | 发送文字 | `/new` |
| 停止 | 发送文字 | `/stop` |
| 帮助 | 发送文字 | `/help` |
| Cursor | 发送文字 | `/backend cursor` |
| Claude | 发送文字 | `/backend claude` |

用户点击后，文字会作为消息发给机器人，码桥按斜杠命令处理。

### 方式 B：推送事件（可自定义响应）

1. 在 **事件与回调** 订阅：
   - `application.bot.menu_v6`（菜单点击）
   - `im.chat.access_event.bot_p2p_chat_entered_v1`（用户打开单聊，发欢迎语）
2. 菜单项动作选 **推送事件**，填写 `event_key`：

| 菜单名称 | event_key |
|---------|-----------|
| 帮助 | `fcb_help` |
| 状态 | `fcb_status` |
| 续聊 | `fcb_resume` |
| 新会话 | `fcb_new` |
| 停止 | `fcb_stop` |
| Cursor | `fcb_backend_cursor` |
| Claude | `fcb_backend_claude` |
| 工作区 | `fcb_ws_list` |

码桥启动后会自动处理上述 `event_key`（见 `packages/router/src/command-help.ts`）。

## 欢迎语

用户**首次**打开与机器人的单聊时，码桥会自动发送欢迎消息和常用命令说明。

## 限制

- 自定义菜单目前仅 **单聊** 生效；群聊请直接 `@机器人` + 斜杠命令。
- 菜单配置保存后约 5 分钟生效；客户端需飞书 7.22+（悬浮菜单）。
