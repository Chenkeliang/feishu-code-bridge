# V2EX 发帖稿

## 使用说明

1. 打开 [V2EX · 分享创造](https://www.v2ex.com/go/create)
2. **标题栏**粘贴：

   ```text
   做了个飞书机器人，路上也能让本机 Cursor / Claude 接着改代码
   ```

3. **正文栏**打开 [v2ex-post.body.md](./v2ex-post.body.md)，全选复制粘贴（已是标准 Markdown，直接粘 V2EX 即可）
4. 建议配一张飞书对话截图

## 发帖提示

| 项 | 说明 |
| --- | --- |
| 节点 | 分享创造 |
| 链接 | 若正文链接触发限制，GitHub 链接放首条回复 |
| 配图 | 一张飞书截图即可 |

## 文件说明

| 文件 | 用途 |
| --- | --- |
| [v2ex-post.body.md](./v2ex-post.body.md) | 正文（复制这一份） |
| 本文件 | 标题 + 发帖说明 |
